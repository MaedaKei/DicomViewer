# CT-Analyzer: DICOM画像解析ツール
## 概要
DICOM画像の表示・解析ができるビューワです。
## セットアップ
### 前提条件
- node.js v22.18.0
- npm 10.9.3
### インストール
1. リポジトリをクローン
```bash
git clone <repository-url>
cd DicomViewer
```

2. 依存関係をインストール
リポジトリをクローンして作成されたディレクトリに移動し、以下のコマンドを実行する。
リポジトリ内のpackage-lock.jsonをもとに必要なパッケージをインストールします。
```bash
npm ci
```
